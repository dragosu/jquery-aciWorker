.call (data, callback) Call a function or method. where: data
  • object {
  • name: (string) the function or method name,
  • data: (mixed) the parameter value(s) (if any)
  • }
 callback function (data) a callback function called at the end of the operation with the data parameter equal with the initial one plus a few more added properties, where data.result is the result of the called function or method. data.name can contain one or more [.] (point) characters to access namespaced elements, data.data can be an array if there are more parameters than one (or an array with a single element - when there is a single parameter but the value itself is an array).
.construct (data, callback) Create object instance. where: data
  • object {
  • name: (string) name of the variable,
  • type: (string) the object type,
  • data: (mixed) the parameter value(s) (if any)
  • }
 callback function (data) a callback function called at the end of the operation with the data parameter equal with the initial one plus a few more added properties. data.name can contain one or more [.] (point) characters to access namespaced elements, data.data can be an array if there are more parameters than one (or an array with a single element - when there is a single parameter but the value itself is an array). The maximum number of parameters is hard coded to #4 (but can be changed in code if more are required).
.destroy () Destroys the aciWorker instance.
.destruct (data, callback) Destroys object instance. where: data
  • object {
  • name: (string) name of the variable
  • }
 callback function (data) a callback function called at the end of the operation with the data parameter equal with the initial one plus a few more added properties. data.name can contain one or more [.] (point) characters to access namespaced elements.
.execute (task, data, callback) Calls one of the API methods, where task is the method name. where: task (string) the API method name data (object) depending of the method callback function (data) a callback function called at the end of the operation with the data parameter equal with the initial one plus a few more added properties.
.get (data, callback) Return the value of a variable. where: data
  • object {
  • name: (string) name of the variable
  • }
 callback function (data) a callback function called at the end of the operation with the data parameter equal with the initial one plus a few more added properties, where data.result is the value of the variable. data.name can contain one or more [.] (point) characters to access namespaced elements.
.ifset (data, callback) Set the value of a variable only when the variable was not defined. where: data
  • object {
  • name: (string) name of the variable,
  • data: (mixed) the value to be set
  • }
 callback function (data) a callback function called at the end of the operation with the data parameter equal with the initial one plus a few more added properties. data.name can contain one or more [.] (point) characters to access namespaced elements.
.init () Init the aciWorker plugin.
.isset (data, callback) Test if a variable was defined or not. where: data
  • object {
  • name: (string) name of the variable
  • }
 callback function (data) a callback function called at the end of the operation with the data parameter equal with the initial one plus a few more added properties, where data.result is the variable state. data.name can contain one or more [.] (point) characters to access namespaced elements.
.load (data, callback) Import one or more script files. where: data
  • object {
  • file: (string/array) file name(s)
  • }
 callback function (data) a callback function called at the end of the operation with the data parameter equal with the initial one plus a few more added properties. The imported script files need to be accesible by the web worker (the `aciWorker.js` script).
.set (data, callback) Set the value of a variable. where: data
  • object {
  • name: (string) name of the variable,
  • data: (mixed) the value to be set
  • }
 callback function (data) a callback function called at the end of the operation with the data parameter equal with the initial one plus a few more added properties. data.name can contain one or more [.] (point) characters to access namespaced elements.
.toString (data, callback) Return the value of a variable, function or method by using the native method toString. where: data
  • object {
  • name: (string) the object name
  • }
 callback function (data) a callback function called at the end of the operation with the data parameter equal with the initial one plus a few more added properties, where data.result is the value returned by toString. data.name can contain one or more [.] (point) characters to access namespaced elements.
.wasInit () Returns TRUE if the plugin has been initialized.
